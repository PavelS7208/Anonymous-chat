# Anonymous-chat

Тестовая задача. Создать анонимный чат по ТЗ

Задание описано в [Постановка](DESC%20task.md )


Данныа задача - учебная. 
Эволюция решений от простой реализации к более сложной до production-ready системы

В ветках  step_N будут размещаться этапы решение задачи.

## Эволюция решений

### Вариант 1. Быстро с минимальной функциональностью, но рабочий вариант

[ReadMe step_1](https://github.com/PavelS7208/Anonymous-chat/tree/step_1)

### Вариант 2. Время подумать, спроектировать архитектуру, но просто работающий вариант

[ReadMe step_2](https://github.com/PavelS7208/Anonymous-chat/tree/step_2)

### Вариант 3. Крупный рефакторинг прошлого варианта. 

Последний готовый вариант [step_3](https://github.com/PavelS7208/Anonymous-chat/tree/step_3)

Реализованы защиты от сетевых атак, логирование (с динамическим измением уровня логирования), тесты.\
Явно выраженная архитектура: http - сервис - домен\
Поддерживаемый код, готовый к расширению и новым фичам с минимальными изменениями.

### Вариант 4. In progress .....

Пред prod-ready 

- Запуск локально и в контейнере
- Сквозной log с traceID в JSON с разными компонентами (ошибки/события инфорструктуры, дебаг домена, .... )
- Рефакторинг доменного слоя. Выделен чистый бизнес-домен, инфраструктура доставки событий и оркестрация работы чата
- Заложены возможности миграции с InMemory репозитария комнат на Redis и/или иные
- Заложены возможности подключения метрик, аудита, статистики

Текущий вариант в работе [step_4](https://github.com/PavelS7208/Anonymous-chat/tree/step_4)
